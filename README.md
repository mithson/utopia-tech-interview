
## To Run Locally 

Clone the project

```bash
  git clone https://github.com/mithson/utopia-tech-interview.git
```

Go to the project directory

```bash
  cd utopia-tech-interview
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
